# 模式化学习总纲

- **输入** -> 模式化学习 -> **输出** -> 反馈 -> **最佳实践**

## 输入

### 来源

1. 专业书籍
2. 视频教程
3. 文章（公众号、技术社区）
4. Github
5. 刷题

### 资源类型

1. 从0到1的技术类图书
2. 视频教程
   - 基础讲解（eg. xxx从入门到精通）
     - 视频总时间长
   - 应用类视频 （eg. 慕课网实战视频）
     - 单套教程20多个小时
     - 学完了直接可以干活
   - 源码类视频
     - 实现框架
     - 一般比较难，需要反复听
