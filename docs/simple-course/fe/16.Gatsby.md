# Gatsby

## 教程

### 安装

安装 gatsby-cli

```shell
npm install gatsby-cli -g
```

查看 gatsby 版本号

```shell
gatsby --version
```

### 创建第一个 gatsby 项目

使用`gatsby new`来创建第一个Gatsby项目

```shell
gatsby new
```

根据cli的提示创建好项目之后

进入项目

```shell
# 进入初始化项目创建的文件夹
cd project

# 启动开发server
npm run develop
```

访问项目

```
You can now view my-first-gatsby-site in the browser.
⠀
  http://localhost:8000/
⠀
View GraphiQL, an in-browser IDE, to explore your site's data and
schema
⠀
  http://localhost:8000/___graphql
```
