# Gatsby

## 教程

### 安装

安装 gatsby-cli

```shell
npm install gatsby-cli -g
```

查看 gatsby 版本号

```shell
gatsby --version
```

### 创建第一个 gatsby 项目

使用`gatsby new`来创建第一个Gatsby项目

```shell
gatsby new
```
