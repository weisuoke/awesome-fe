# Vue3

## API

### createApp

返回一个提供应用上下文的应用实例。应用实例挂载的整个组件树共享同一个上下文。

```js
const app = Vue.createApp({})
```

你可以在 `createApp` 之后链式调用其它方法，这些方法可以在[应用 API](https://vue3js.cn/docs/zh/api/application-api.html) 中找到。
