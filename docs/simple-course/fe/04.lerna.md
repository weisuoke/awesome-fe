# Lerna

## 安装

```shell
npm install --global lerna
```

接下来，我们将创建一个新的 git 代码仓库：

```shell
git init lerna-repo && cd lerna-repo
```

现在，我们将上述仓库转变为一个 Lerna 仓库：

```shell
lerna init
```
