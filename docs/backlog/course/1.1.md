# 1.1 课程导学

为什么学算法、读源码，工作中却找不到应用点？怎么能到架构师？

- 因为逻辑链反了，应该是
  - 遇到问题
  - 产生需求，才能驱动更深的技术
  - 比如算法、设计模式



做过架构师难度的项目，去应聘架构师，而不是死学技术等着架构师岗位找上门。

你需要一个这样的项目

- 够难度
- 够复杂
- 够锻炼水平
- 够吸引人
- 大厂标准
